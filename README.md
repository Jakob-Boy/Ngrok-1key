# Ngrok 内网穿透一键配置脚本

使用Ngrok的隧道，一键转发内网的Http/TCP端口

> PS: 由于Ngrok免费版本的限制，本脚本只能转发HTTP和TCP协议

## 使用方法

```shell
wget -N https://raw.githubusercontents.com/Misaka-blog/Ngrok-1key/master/ngrok.sh && bash ngrok.sh
```

快捷方式 `bash ngrok.sh`

## 使用说明

1. 在 https://ngrok.com/ 注册一个账号，获取Ngrok Authtoken
2. 使用本脚本
